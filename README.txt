타임루트 AI v4 배포용

수정 사항:
1. 전체 디자인 재구성
   - 상단 여백 과다 문제 개선
   - 지도/카드 영역 비율 재정리
   - 카드/버튼/입력창 스타일 재정리
   - 반응형 개선

2. 랜덤 루트 로직 강화
   - data/route_engine.json 기반 카테고리/프로필/설명 엔진 추가
   - 전통시장, 맛집, 카페, 간식, 포토스팟, 문화, 산책, 상권 키워드 검색
   - Google Places type 검색 + keyword 검색 병행
   - 장소 다양성, 거리, 평점, 리뷰 수, 랜덤 가중치 반영
   - 루트 프로필 8종 적용
   - 루트별 제목/설명 중복 최소화
   - 장소 순서 최적화로 이동거리 과도 증가 방지
   - 입력 시간 초과 시 장소 수 자동 조정

3. 유지 기능
   - 현재 위치 사용
   - 지역명 직접 입력
   - 랜덤 루트 5개
   - 다시 뽑기 무제한
   - 루트 카드 클릭 시 설명/지도 경로선 변경
   - Google 지도 길찾기 연결

배포:
ZIP 압축 해제 후 내부 파일 전체를 GitHub Pages 저장소에 업로드하세요.
