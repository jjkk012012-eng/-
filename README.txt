타임루트 AI v5

수정 핵심:
- 첫 화면 문구 절반 이하로 축소
- 긴 설명 대신 키워드 중심 문구로 변경
- 루트 카드에 실제 추천 가게/장소 이름을 크게 표시
- 각 장소마다 번호와 유형 표시
- 건물/아파트/정류장 등 불필요한 장소명 필터 강화
- 맛집/카페/시장/간식/명소/산책/상권 중심 검색 강화
- 카드 클릭 시 지도 마커 title에도 실제 가게/장소 이름 표시
- 지도 헤더에 대표 장소명 3개 표시
- 기존 기능 유지: 현재 위치, 직접 입력, 루트 5개, 다시 뽑기, 지도 경로선

배포:
ZIP 압축 해제 후 내부 파일 전체를 GitHub Pages 저장소에 업로드하세요.


v6 수정 사항:
- 상단 카피를 '내 주변을 랜덤 코스로'로 변경
- '시간만 고르면 갈 곳 5개 추천' 문구 제거
- 상단 레이아웃 균형 재조정
- 히어로/검색 카드 간격과 지도 시작 위치 개선
- 핵심 키워드형 문구 유지

v7 수정 사항:
- 상단 입력 카드와 지도/추천 카드가 겹치던 레이아웃 오류 수정
- workspace의 음수 margin 제거
- 지도 영역과 추천 카드 영역을 상단 섹션 아래로 분리
- 모바일/태블릿에서도 겹치지 않도록 margin 재조정
- 지도 오류가 나도 카드 레이아웃이 겹치지 않도록 높이 조정

참고:
지도에 '죄송합니다. 문제가 발생했습니다'가 보이면 레이아웃 문제가 아니라 Google Maps API 키 제한/활성화 문제입니다.
Google Cloud Console에서 Maps JavaScript API, Places API, Directions API, Geocoding API 활성화와 HTTP referrer 도메인 허용을 확인하세요.

v8 수정 사항:
- 검색 카드 안 보조 문구가 흰색으로 보이는 문제 완전 수정
- search-card 내부 텍스트 색상을 명시적으로 검정/회색으로 고정
- 상단/지도/추천 카드의 삐뚤어진 레이아웃 재정렬
- 음수 margin 제거 및 전체 max-width 통일
- 히어로 카피/검색 카드/지도 카드의 좌우 균형 재조정
- 모바일/태블릿 겹침 방지

v9 수정 사항:
- 1→2→3→4 경유지 순서를 최대한 짧게 이어지도록 재정렬
- 최근접 탐색 + 2-opt 개선 로직 추가
- 지도에는 직선이 아니라 Google Directions의 실제 도보 경로선 표시
- Directions 실패 시에만 임시 polyline 표시
- Directions 결과의 실제 거리/이동시간을 상단 요약에 반영
